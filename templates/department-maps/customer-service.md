## AGENT CONFIGURATION NOTES

AUTOMATED (no approval needed):
→ Inbound enquiry response (immediate acknowledgment + ETA)
→ FAQ answers via VAPI or WhatsApp
→ Booking confirmations and reminders
→ Review request sent after job completion (24hr delay)
→ Win-back message to customers inactive 30+ days
→ Positive review response drafts (posted immediately)
→ Negative review response drafts (held for approval)

REQUIRES APPROVAL:
→ Refund authorisation
→ Response to serious complaints (legal risk)
→ Removal request for negative reviews
→ Any customer escalation to owner

MONITORED (reported weekly):
→ Response time per channel (vs target)
→ Open complaints (unresolved > 48hrs)
→ New reviews (positive and negative)
→ Churn rate this month
→ Win-back campaigns sent and conversion rate
→ NPS score (if tracked)
