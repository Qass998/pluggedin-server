# Huashu Design System

Read and apply the full design system before generating any visual output.

**Action:** Read `skills/huashu-design/SKILL.md` in full, then confirm which anti-slop rules and design philosophies apply to the current task. Apply them to every visual decision without exception.

**Rules enforced:**
- NO purple gradients
- NO emoji icons
- NO Inter as display font (use Syne or similar)
- USE oklch() colors
- USE CSS Grid over flexbox
- USE text-wrap: pretty
- ONE accent color maximum

**Triggers automatically for:** any HTML, dashboard, report, website, or UI generation task.
