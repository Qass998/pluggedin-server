# Dashboard Builder

Generate a PluggedIN OS dashboard or client demo.

**Action:**
1. Read `skills/huashu-design/SKILL.md` first
2. Then read `skills/dashboard/SKILL.md`
3. Follow all steps in the dashboard SKILL.md
4. Save output to `outputs/dashboards/demo_{industry}_{date}.html`

**Triggers for:** "build a dashboard", "generate a demo", "create a KPI tracker", "build a client demo", "show me the [module] dashboard"
