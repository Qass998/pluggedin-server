# Morning Briefing

Deliver the daily PluggedIN operating briefing.

**Action:**
1. Read `memory/working/today.md`
2. Read `memory/working/pipeline.md`
3. Read `memory/episodic/log.md` (last 20 lines)
4. Deliver briefing in standard format:

SITUATION: [facts and numbers, 3 sentences max]
PRIORITY: [single most important thing]
RECOMMENDATION: [specific action, precise enough to execute]
REASONING: [why this, 2 sentences]
EXPECTED OUTCOME: [what happens if we act today]
PROCEED? [yes / no / adjust]

Never ask what to do. Read data. Tell Qassim what's happening.
