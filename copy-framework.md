## TONE GUIDE

DO:
→ Be direct and specific
→ Reference their actual work
→ Make a clear valuable connection
→ Respect their time
→ Sound like a person not a template

DO NOT:
→ Use corporate filler language
→ Mention AI or automation
→ Over-explain PluggedIN
→ Pitch Gromatic's services in email 1
→ Use more than one CTA
→ Write more than 100 words in email 1

---

## QUALITY CHECK

Before sending any email run this check:

→ Is the personalisation line specific? (not generic)
→ Is it under 100 words?
→ Does it send as PluggedIN not Gromatic?
→ Is there exactly one CTA?
→ Does it reference a real signal?
→ Does it sound human?
→ Would you be embarrassed if Damian read it?

If any answer is no: rewrite before sending.
