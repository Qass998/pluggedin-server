# PluggedIN Live — Weekly Operating Routine
# Runs: Every Monday at 06:00 (before daily routine)
# Purpose: Strategic review. Scale what's working. Kill what isn't.
#          Find the next move. Keep compounding.

---

## MONDAY — WEEKLY STRATEGY BRIEFING

Delivered via WhatsApp at 07:00 (replaces/extends daily briefing).
Longer format. Qassim reviews before the week begins.

---

### WHAT THE WEEKLY BRIEFING COVERS

REVENUE REVIEW (last 7 days):
→ Total revenue across all Live businesses
→ Revenue per business (vs last week, vs target)
→ Best performing vertical / product / deal
→ Biggest miss — why, and what's the fix

PIPELINE HEALTH:
→ Lead Gen: buyers added, leads in flow, invoice collection rate
→ Ecommerce: active products, ROAS trends, new launches queued
→ AgriTrade: deals by stage, largest pending deal, stalled deals
→ YouTube: subscribers gained, monetisation progress, best video

OPPORTUNITY SCAN (last 7 days):
→ Top 3 opportunities Opportunity Engine flagged
→ Any that scored ≥ 70 not yet actioned — why not?
→ Recommendation: Launch / Monitor / Pass for each

KNOWLEDGE UPDATE (what agents learned last week):
→ Top 3 insights from Knowledge Agent
→ Any skill gaps identified
→ Any copy framework updates made
→ Any ICP profile updates

DECISIONS FOR THE WEEK:
→ What needs Qassim's input this week (not today — the week)
→ Presented as numbered list with recommendation per item

## TUESDAY — CONTENT BATCH DAY

All 5 YouTube channels get their weekly research done on Mondays.
Tuesday: scripts finalized and production queued.
Agent works through the content pipeline in sequence.

Produced by EOD Tuesday:
→ 5 video scripts (one per channel) — finalized and approved
→ 5 voiceovers generated (ElevenLabs — one per voice profile)
→ 5 video outlines in Creatomate (ready for assembly Wednesday)

---

## WEDNESDAY — PRODUCTION AND OUTREACH PUSH

YouTube: All 5 videos assembled and scheduled for publishing.

Lead Gen: Mid-week outreach push
→ 10 new buyer prospects per vertical (50 total)
→ Follow-up emails on anyone who opened but didn't reply
→ Check all delivered leads — any payment outstanding?

AgriTrade: Mid-week deal nudge
→ Any deal in "Negotiating" stage for >5 days → nudge email to both
→ New producer VAPI calls (batch run)
→ LinkedIn outreach to 10 new buyer prospects

---

## THURSDAY — REVIEW SCORES AND REFORECAST

Ecommerce: Mid-week ad review
→ Pull ROAS data for all active dropshipping products
→ Scale anything at ≥ 2.5x ROAS (increase daily budget 20%)
→ Pause anything at < 1.5x ROAS after 5+ days
→ Check Gumroad/Etsy sales — any products spiking?
→ Check Pinterest analytics — any pins going viral?

Opportunity Engine: Mid-week rescore
→ Rescore any "Monitor" opportunities from last 7 days
→ If score moved to ≥ 70, add to Friday briefing
→ Log final scores and decisions

---

## FRIDAY — WEEKLY PERFORMANCE LOG AND NEXT WEEK PREP

KPI Agent:
→ Pull all revenue data from Airtable
→ Compile weekly performance report per business
→ Compare vs targets
→ Flag what's ahead of target (scale it) vs behind (diagnose it)
→ Save to outputs/reports/week-[N]-performance.md

Narrative Agent:
→ Generate weekly Remotion summary video (optional — monthly is priority)

Chief of All Chiefs:
→ Prepare Monday's Weekly Strategy Briefing early (ready for 06:00 Monday)
→ Identify this week's top insight and next week's one big move

---

## MONTHLY (1ST OF EVERY MONTH)

Narrative Agent:
→ Full Remotion portfolio narrative video (all 5 businesses, 90 seconds each)
→ Written monthly report — revenue, growth, decisions, pipeline
→ Save to outputs/reports/month-[YYYY-MM]-portfolio.md

Chief of All Chiefs:
→ Cross-business CEO meeting (agent-to-agent synthesis)
→ Revenue vs targets vs Month 1 roadmap
→ Businesses to scale up / scale down / pause
→ Opportunities to activate this month
→ One strategic recommendation for Qassim

Knowledge Agent:
→ Monthly compression of all KnowledgeLog entries
→ Archive insights older than 30 days to semantic memory
→ Update ICP, copy framework, and scoring criteria based on 30-day learnings

Maintained in Airtable: CompoundTracker table
Updated weekly. Shows:

| 1    | £0      | 0          | 0     | 0      | 0           | 0     |
| 2    | ...     | ...        | ...   | ...    | ...         | ...   |

This table answers the question: is the system compounding?
If revenue per business is growing week on week, the system works.
If it's flat: Knowledge Agent investigates, CEO Agent recommends fix.
The answer is always in the data.
